Layer inspector

Select any layer(s)—frames, groups, components, vectors, etc.—then refresh to see the tree. Position (x, y) is relative to each layer’s parent’s top-left; size, visibility/lock, and text details are included when available. The Struct tab exports Aura Post Card JSON: select a root container that has a visible child named front (required). Optional children: back, inside, colorOptions.
